# Contributing to AquaTutor

AquaTutor is an open-source research project exploring semi-supervised underwater fish detection.

## Project Principles

- Build incrementally.
- Keep work reproducible.
- Document all major decisions.
- Use clear notebook names.
- Record experiments carefully.
- Keep large datasets and trained models out of GitHub unless explicitly required.

## Notebook Naming

Use numbered notebooks:

01_Environment_Setup.ipynb
02_Dataset_Acquisition.ipynb
